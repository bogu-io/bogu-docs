# Command Line Options

All the bells and whistles.

## Usage

### --help, -h

Show help text.

## <option> is one of

### -p &lt;PATH&gt;, --path &lt;PATH&gt;

Local scan path.

### --github-owner &lt;OWNER&gt;

GitHub Repo Scan by Owner

## Mutually Exclusive Options

### --version

Prints bogu version.

### --symlink

Creates a symbolic link in `/usr/local/bin`.
